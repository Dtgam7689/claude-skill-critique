## 🧰 What this tool does

claude-skill-critique is a Claude Code skill for adversarial review of plans, design docs, and code.

It helps you spot weak points before they become real problems. It can review text for gaps, bad assumptions, missing steps, unclear logic, and code that may not hold up under pressure.

Use it when you want a hard-nosed review before you move forward.

## ✨ What you can use it for

- Review project plans before work begins
- Check design docs for missing details
- Find weak logic in written proposals
- Review code for edge cases and risk
- Stress test ideas before you commit time to them
- Get a second pass on work that feels too confident
